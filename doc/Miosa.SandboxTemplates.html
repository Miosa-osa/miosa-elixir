    Miosa.SandboxTemplates (Miosa v1.0.2)

    Copy Markdown View Source

    Sandbox template management — define reusable base images for sandboxes.

    Templates are built from a build_spec (a declarative definition of the base image). Use build_spec_schema/1 to discover the schema, validate/2 to check a spec before creating a template, and create_build/2 to trigger a build.

    Mutating calls (create, create_build) send an Idempotency-Key header automatically.

    Example

    client = Miosa.client(System.fetch_env!("MIOSA_API_KEY"))

{:ok, schema} = Miosa.SandboxTemplates.build_spec_schema(client)

{:ok, tmpl} = Miosa.SandboxTemplates.create(client, %{
  name: "node-20-base",
  build_spec: %{runtime: "node", version: "20", packages: ["curl"]}
})

{:ok, build} = Miosa.SandboxTemplates.create_build(client, tmpl["id"], %{})
